# 💰 FinTrack Pro - Expense Analytics Dashboard

FinTrack Pro is a modern and interactive Expense Tracking Dashboard built using HTML, CSS, and JavaScript. It helps users manage daily expenses, analyze spending patterns, and visualize financial data through charts and analytics.

## 🚀 Features

* Add and manage expenses easily
* Categorize expenses for better tracking
* Daily, Weekly, and Monthly expense summaries
* Interactive Doughnut Chart visualization
* Transaction history table
* Delete expenses with smooth animations
* Data persistence using Local Storage
* Responsive and modern glassmorphism UI
* Real-time expense analytics

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js
* GSAP Animation Library
* Local Storage API
* Font Awesome Icons

## 📊 Dashboard Modules

### Expense Entry

* Expense Name
* Amount
* Date
* Category Selection

### Analytics Section

* Daily Total Expenses
* Weekly Total Expenses
* Monthly Total Expenses
* Category-wise Expense Distribution Chart

### Transaction History

* View all recorded expenses
* Category badges
* Delete transactions

1. Download the repository.
